QUE:-1 Given an array const a = [1, 3, 5, 2, 4] generate an array result1 from a, which should be equal to [1, 5, 4]. The solution should be of the form: 

ANS:- const a = [1, 3, 5, 2, 4];

const result1 = a.filter(x => x % 2 === 1);

console.log(result1);
